# GooGooGaaGaaLang
The world's best language.

## Features
- **Binary-based**: Everything is `goo` and `gaa`.
- **Emotional Debugging**: Errors that scream at you in `rAnDoM cAsE`.

## Commands
- `uuu`: 1s pause (Baby is thinking).
- `hehehe`: Clears terminal (Baby is happy).
- `mmm`: Silent input (Baby is waiting).
